# Geometry-Aware High-Frequency Feature Extraction from Monocular Images

## Setup

```bash
uv sync
```

Then download the DTD (Describable Textures Dataset) images, used by
`main.py` and the validation scripts:

```bash
mkdir -p dtd_data
cd dtd_data
curl -L -o dtd.tar.gz "https://thor.robots.ox.ac.uk/dtd/dtd-r1.0.1.tar.gz"
tar xzf dtd.tar.gz dtd/images
rm dtd.tar.gz
```

## DSINE checkpoint

Not tracked in git (278MB, over GitHub's 100MB push limit, and it's a
third-party artifact anyway). To get it:

1. Go to the official checkpoints folder (linked from
   https://github.com/baegwangbin/DSINE, maintained by the paper authors):
   https://drive.google.com/drive/folders/1t3LMJIIrSnCGwOEf53Cyg0lkSXd3M4Hm
2. Open `checkpoints/exp001_cvpr2024/` and download only `dsine.pt` (the
   "best model" from the paper; ignore `exp002_kappa/`, not used here).
3. Save it to `external/dsine/checkpoints/exp001_cvpr2024/dsine.pt`.

## Project layout

- `src/dsine_infer.py` — device-agnostic (CPU/MPS) loader and inference wrapper for DSINE.
- `src/wavelet_energy.py` — the DWT/WEGE-style wavelet energy map.
- `src/fusion.py` — the normal-variation map and the explicit fusion rule.
- `external/dsine/` — vendored (copied and lightly patched) DSINE model code.
- `dtd_data/` — downloaded separately (see Setup above); used both for the qualitative demo (`main.py`) and the quantitative validation.
- `outputs/` — everything generated by the scripts below.

## Scripts

- **`main.py`** — qualitative demo. Runs the pipeline on a few chosen DTD images, saves a 4-panel figure per image (original, wavelet-only attention, DSINE-only attention, fused attention) to `outputs/<name>_pipeline.png`. Seconds to run.
- **`analysis.py`** — quantitative validation. Runs the pipeline over ~5,000 DTD images (47 categories bucketed into flat vs. real-3D-relief) and saves per-image stats to `outputs/dtd_analysis.csv`. ~12-15 min.
- **`roc_analysis.py`** — reads `dtd_analysis.csv` (run `analysis.py` first) and computes ROC/AUC for wavelet-only, DSINE-only, and combined signals, saving `outputs/roc_curves.png`. Seconds to run.

## Recommended order

1. `uv sync` + download the checkpoint + download DTD (above).
2. `uv run python main.py`
3. `uv run python analysis.py`
4. `uv run python roc_analysis.py`
